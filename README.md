# Income & Expense Telegram Bot

Telegram-бот для учёта личных доходов и расходов. Позволяет быстро добавлять операции, просматривать статистику за любой период, редактировать или удалять записи, а также строить наглядные графики по дням.
## Описание

- **Добавление доходов и расходов** – две отдельные кнопки для быстрого ввода.
- **Редактирование записей** – изменение типа (доход/расход), суммы или полное удаление по ID.
- **Статистика за период** – выбор начальной и конечной даты через удобный календарь.
- **Детализация за день** – при выборе одного дня показывается список всех операций с их ID.
- **График доходов/расходов** – для диапазона от 7 до 31 дня строится столбчатая диаграмма.
- **Мгновенные подтверждения** – после добавления операции отправляется картинка-реакция.
- **Ограничение доступа** – бот работает только для одного указанного пользователя (по Telegram ID).

## Стек

- **Python 3.10+**
- **aiogram 3.x** – асинхронный фреймворк для Telegram Bot API
- **SQLite** – база данных (с возможностью лёгкой замены на PostgreSQL)
- **matplotlib** – построение графиков
- **aiogram-calendar** – интерактивный календарь для выбора дат
- **python-dotenv / config.py** – управление конфигурацией

## Установка и запуск

1. Склонировать репозиторий

```
git clone https://github.com/justtryhard/income-expense-bot.git
cd income-expense-bot
```


2. Создать виртуальное окружение

Windows:

```
python -m venv venv
venv\Scripts\activate
```

Linux / macOS:

```
python3 -m venv venv
source venv/bin/activate
```

3. Установить зависимости

```
pip install -r requirements.txt
```

4. Сконфигурировать окружение

Заменить значения в config.py:

```
API_TOKEN = "YOUR_TELEGRAM_BOT_TOKEN"
DB_NAME = "bot.db"
ALLOWED_USER_ID = "YOUR_USER_ID"
```


5. Запустить

```
python main.py
```
